Instructions

This is a turn tracker for the board game Mansions of Madness 2nd Edition.

To use, click the Add Player button as many times as desired. Then you can click each player name to change it.

Then simply click either the Action or Move button for a player. The purple arrows next to a player represent actions. Pressing the Action button will use one action, which will cause it to turn gray.

Pressing the Move button with no moves available will turn one available action into two moves, represented by smaller gold arrows. One will be used immediately.

If no more moves or actions are available a message telling you so is printed to the message box. Other messages are printed on certain actions and remain in the message box until the page is reloaded.

Click the New Turn button to restore each players actions. You can also click Reset on each player to restore that player's actions.
